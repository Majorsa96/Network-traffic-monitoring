# Network Traffic Monitoring Using Wireshark
This repository contains a network traffic capture and analysis using the wireshark tool.

## Files
- `Network_Traffic_Analysis.pcapng` - Raw network traffic capture
- `Network_Traffic_Monitoring_Report.md` - Detailed Monitoring Report
- `Screenshots/` - Optional Wireshark screenshot

## How to open
1. Install Wireshark
2. Open `Network_Traffic_Analysis.pcapng` to view the captured traffic
3. Refer to the report for observation and analysis
